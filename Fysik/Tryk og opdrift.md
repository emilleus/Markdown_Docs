# Opdrift

## Krafter

**Kraft:**
* Symbol = $F$
* Enhed = $N$

**Tryk:**
* Symbol = $p$
* Enhed = $Pa$

når man tegner krafter ind på ting, så tegner man "kraftpile", som er det samme som vektorer. Den skal sidde på masse-midtpunktet. 

Normalkraften skal starte der hvor de 2 flader mødes.

* $F_t$ = tyngdekraft
* $F_N$ = normalkraft 
* $F_{træk}$ = trækkraft
* $F_μ$ = friktionskraft.

$$p = \frac{F}{A}$$

* $A$ = areal
